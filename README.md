## Setup

Install [nodejs and npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

Tested on...

node - 20.14
npm  - 11.2.0

## Getting Started

Install deps

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
